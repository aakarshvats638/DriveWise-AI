# 🚗 DriveWise AI

DriveWise AI is a full-stack car recommendation platform that helps users discover suitable cars based on their preferences and compare shortlisted vehicles side by side.

# What I Built

I built a full-stack web application with two main user journeys:

### 1. Find Me a Car
Users can enter:
- Maximum Budget
- Fuel Type
- Transmission
- Body Type

The system recommends the most suitable cars using a rule-based recommendation engine and displays recommendation reasons.

### 2. Compare My Shortlist
Users can select multiple cars and compare them based on:
- Price
- Fuel Type
- Transmission
- Mileage
- Safety Rating
